# 국가별 가이드

각 국가 README는 그 나라의 **모든 도시에 공통으로 적용되는 사실의 정본**이다 — 통화·콘센트·시차·비자·통신·결제·응급 연락처 같은 것. 도시별로 갈리는 정보(공항·숙소·교통·식당·치안)는 도시 문서로 분기된다.

새 국가/도시는 [../templates/](../templates/)에서 복사해 시작한다. 형식·규칙은 [../AGENTS.md](../AGENTS.md)와 [../CONTRIBUTING.md](../CONTRIBUTING.md) 참고.
